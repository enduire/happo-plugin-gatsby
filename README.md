# happo-plugin-gatsby

A [happo.io](https://github.com/enduire/happo.io) plugin for Gatsby.

## Usage

Add the following to your `.happo.js` configuration file:

```js
// .happo.js
const happoPluginGatsby = require('happo-plugin-gatsby');

module.exports = {
  // ...
  plugins: [
    happoPluginGatsby({
      // options go here
    }),
  ],
}
```
